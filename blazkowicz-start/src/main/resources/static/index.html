使用说明

  • 在「事件管理」创建事件，配置 Feature 特征与 StrategyPack 策略包。
  • 先在「规则配置」创建 Groovy 规则，再在「策略配置」绑定规则与处置。
  • 「运行态测试」会读取当前生效的策略，实时执行规则引擎。
  • 默认 TEST 场景 seed 规则：user_id=114515 时条件全为 true，返回 NOT_MEET；其他 user_id 返回 MEET 并触发处置。
